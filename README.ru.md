# Screenshot Manager

[English](README.md) · [Русский](README.ru.md) · [简体中文](README.zh-CN.md)

Нативный менеджер скриншотов для macOS: быстрый захват экрана, аннотации, работа через буфер обмена и локальная библиотека снимков.

![Вид библиотеки](docs/screenshots/library.png)

## Возможности

- Глобальные хоткеи для захвата в буфер обмена и в библиотеку
- Оверлей для захвата области экрана или окна
- Плавный редактор аннотаций с кадрированием, изменением размера, стрелками, фигурами, пером, текстом, размытием и фонами
- Пипетка цвета и координаты позиции прямо в оверлее захвата
- Быстрый clipboard-first сценарий для мгновенных скриншотов
- Отдельный сценарий сохранения в локальную библиотеку
- Поиск по локальной библиотеке скриншотов
- Быстрый предпросмотр, копирование, редактирование, показ в Finder и удаление
- OCR-распознавание текста с возможностью копирования
- Нативное окно настроек для хоткеев, разрешений, запуска при входе и отображения иконки в menu bar
- Локальная работа без телеметрии

## Требования

- macOS 14 или новее
- Xcode 16 или новее для локальной сборки
- Разрешение Screen Recording для захвата экрана

Некоторые функции, например распознавание текста, используют нативные Apple frameworks, доступные в macOS.

## Сборка

Откройте `ScreenshotManager.xcodeproj` в Xcode или соберите проект из терминала:

```bash
xcodebuild -project ScreenshotManager.xcodeproj -scheme ScreenshotManager -configuration Debug build
```

## Установка локальной debug-сборки

Скрипт собирает приложение, устанавливает его в `/Applications/Screenshot Manager.app`, подписывает локальным development-сертификатом и открывает:

```bash
./scripts/install-debug-app.sh
```

Так разрешения macOS остаются привязанными к установленному приложению, а не к случайным сборкам из DerivedData.

## Создание DMG

```bash
./scripts/build-dmg.sh
```

Готовый DMG будет создан здесь:

```text
dist/Screenshot Manager.dmg
```

Для публичного распространения приложение нужно подписать и notarize через Apple Developer ID certificate. Локальный signing helper предназначен только для разработки.

## Разрешения

Screenshot Manager требует Screen Recording permission для захвата экрана. Приложение проверяет разрешение перед началом захвата и открывает нужную страницу System Settings, если доступ не выдан.

## Приватность

Скриншоты остаются локально. В приложении нет аналитики, телеметрии, удаленного логирования или загрузки скриншотов на сервер.
